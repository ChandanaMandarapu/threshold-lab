# threshold-lab

A from-scratch Rust playground for Shamir Secret Sharing and toy threshold
signature simulation. Built as daily-commit practice — not production code.

## Why
Reinforces the math/engineering behind threshold signature schemes (TSS):
polynomial interpolation, secret splitting/reconstruction, share validation,
and mock multi-party signing flow.

## Status
🚧 Work in progress — building incrementally, one concept per commit.
